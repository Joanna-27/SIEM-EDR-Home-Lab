# SIEM & EDR Home Lab

## Overview

A hands-on cybersecurity home lab built using **Kali Linux**, **Elastic Stack**, and **Elastic Defend** to gain practical experience in security monitoring, log analysis, and incident investigation.

## Tools Used

- Kali Linux
- Elastic Stack
- Kibana
- Elastic Agent
- Elastic Defend
- VirtualBox

## Key Activities

- Configured a SIEM and EDR environment.
- Monitored endpoint activity using Kibana.
- Investigated security alerts generated from simulated attacks.
- Analyzed events using the MITRE ATT&CK framework.

## Detections Demonstrated

- Hosts File Modified
- Interactive Shell Launched
- Cron Job Created
